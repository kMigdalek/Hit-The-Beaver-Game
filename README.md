# Hit The Beaver Game

Simple browser game created with HTML, CSS and JavaScript.

## Play Online

You can play the game here:

https://kmigdalek.github.io/-Hit-The-Beaver-Game/

## Run Locally

1.Clone the repository

git clone https://github.com/kmigdalek/-Hit-The-Beaver-Game.git

2. Open the project folder

cd -Hit-The-Beaver-Game

3. Open the file

index.html

in your web browser.

## Controls

Use your mouse or keyboard (depending on the game mechanics) to interact with the game and hit the beaver.

## Technologies

- HTML
- CSS
- JavaScript
